# Website Assignment

## Project Title
Simple Website Project

## Project Description
This project is a simple responsive website created as part of a web development assignment. 
The website includes multiple pages designed using HTML, CSS, and Bootstrap. 
It demonstrates basic website layout, navigation, and responsive design.

## Technologies Used
- HTML5
- CSS3
- Bootstrap

## Steps to Run the Project Locally

1. Download or clone the repository.
2. Open the project folder on your computer.
3. Locate the `index.html` file.
4. Double click the `index.html` file.
5. The website will open in your web browser.
